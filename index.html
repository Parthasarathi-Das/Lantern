Here is quick guide for you to know function of each button:
  • Uniform: Clicking this once makes all lanterns glow uniformly together in same color.
  • Random: Clicking this once makes all lanterns glow randomly together in different colours.
  • Shift: Clicking this once makes all lanterns glow one by one and shift colours over time.
  • Creep :Clicking this once makes all lanterns glow one by one in a slow manner as if the brightness creeps through lights.
  • Fusion :Clicking this once makes all lanterns glow one by one like a mixture of oher four.
Remember: Before Clicking any button make sure you have clicked the previous one twice for turning off the effect. If you do so please refresh again.
